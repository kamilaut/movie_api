Movie API Documentation page

This API is a small movie databse with some essential features.

Usage:

To return a list of ALL movies to the user
Method: GET < URL: /movies >
Request Body Data

none

Response

An array of movie records

To return data (description, genre, director, whether it’s featured or not) about a single movie by title
Method: GET < URL: /movies/:Title >
Request Body Data

none

Response

JSON object containing information about a single movie. Example:

                    
  {
      "Genre": {
          "Name": "...",
          "Description": "..."
      },
      "Director": {
          "Name": "...",
          "Bio": "...",
          "Birth": "...",
          "Death": "..."
      },
      "Actors": [],
      "_id": "635bcde5074a7685611cd62e",
      "Title": "Autumn Sonata",
      "Description": "...",
      "ImagePath": "...",
      "Featured": true
  }
To return data about a genre by name/title (e.g., “Thriller”)
Method: GET < URL: /movies/genres/:genreName >
Request Body Data

none

Response

JSON object containing records of the movies by the requested genre. Example:

                    
  {
      "Name": "Crime",
      "Description": "A crime film is a type of film
    focusing on the lives of criminals."
  }
To return data about a director by name
Method: GET < URL: /movies/directors/:directorName >
Request Body Data

none

Response

JSON object with details about the director requested. Example: 

                    
  {
      "Name": "Andrey Zvyagintsev",
      "Bio": "Andrey Petrovich Zvyagintsev is a Russian
      film director and screenwriter.",
      "Birth": "1965",
      "Death": " "
  }
For new users to register
Method: POST < URL: /users >
Request Body Data

JSON object holding data about the users to add.

Response

JSON object with the registered user info. Example

                    
  {
                    "name":"Kamila",
                  "FavoriteMovies": "XYZ",
                "Birthday": "1990-02-06",
              "Email": "kamila@gmail.com"
  }
To allow users to update their user info (username, password, email, date of birth)
Method: PUT < URL: /users/:Username
Request Body Data

none

Response

JSON object with the updated user details. Example 

                        
      {
        "Name":"Kamila",
        "FavoriteMovies": "XYZ",
        "Birthday": "1990-02-06",
        "Email": "kamila@gmail.com"
      }
To add a movie to user's list of favorites
Method: POST < URL: /users/:Username/movies/:MovieID >
Request Body Data

none

Response

List of the user's updated favourite movies. Confirmation that the movie was added into the user's list. Example: 

                    
  {
      "_id": "635c10dae2287d83a9ca369d",
      "Username": "Kamila",
      "Password": "12354",
      "Email": "kamila@gmail.com",
      "Birthday": "1992-07-07T11:30:05.000Z",
      "FavoriteMovies": [
          "635bd79045eec90be7d316e3",
          "635bd75145eec90be7d316e2",
          "635ae0aad3212ebd098dd5ef"
      ]
  }
To remove a movie from user's list of favorites
Method: DELETE < URL: /users/:Username/movies/:MovieID >
Request Body Data

none

Response

List of the user's updated favourite movies. Confirmation that the movie was removed from the user's list. Example: 

                    
  {
      "_id": "635c10dae2287d83a9ca369d",
      "Username": "Kamila",
      "Password": "12354",
      "Email": "kamila@gmail.com",
      "Birthday": "1992-07-07T11:30:05.000Z",
      "FavoriteMovies": [
          "635bd79045eec90be7d316e3",
          "635bd75145eec90be7d316e2",
      ]
  }
For existing users to deregister
Method: DELETE < URL: /users/:Username >
Request Body Data

none

Response

Confirmation that the user has been removed. Example: 

                    
  Kamila was delited.
