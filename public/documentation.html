Movie API Documentation page

This API is a small movie databse with some essential features.

Usage:

To return a list of ALL movies to the user
Method: GET < URL: /movies >
Request body: None. Response: An array of movie records
To return data (description, genre, director, whether it’s featured or not) about a single movie by title
Method: GET < URL: /movies/:Title >
Request body: None. Response: JSON object containing the record for the movie requested
To return data about a genre by name/title (e.g., “Thriller”)
Method: GET < URL: /movies/genres/:Name >
Request body: None. Response: JSON object containing records of the movies by the requested genre
To return data about a director by name
Method: GET < URL: /movies/directors/:Name >
Request body: None. Response: JSON object with details about the director requested
For new users to register
Method: POST < URL: /users >
Request body: JSON object {"name":"Kamila", "favoriteMovies": "XYZ", "Birthday": "1990-02-06", "Email": "kamila@gmail.com"}. Response: JSON object with the registered user info
To allow users to update their user info (username, password, email, date of birth)
Method: PUT < URL: /users/:id
Request body: JSON object {"name":"Kamila", "favoriteMovies": "XYZ", "Birthday": "1990-02-06", "Email": "kamila@gmail.com"}. Response: JSON object with the updated user details
To add a movie to user's list of favorites
Method: PUT < URL: /users/:id/:movieTitle >
Request body: None. Response: List of the user's updated favourite movies. Confirmation that the movie was added into the user's list
To remove a movie from user's list of favorites
Method: DELETE < URL: /users/:id/:movieTitle >
Request body: None. Response: List of the user's updated favourite movies. Confirmation that the movie was removed from the user's list
For existing users to deregister
Method: DELETE < URL: /users/:Username >
Request body: None. Response: Confirmation that the user has been removed
